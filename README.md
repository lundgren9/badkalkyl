# Badanläggningskalkyl – Simrishamns kommun

Webbaserad kalkyl för badanläggningar inom ansvar **40100** (fritidsverksamhet), verksamhet **3400**, med utgångspunkt i **Kommun-BAS 26**. Applikationen hanterar Tobisviksbadet, Hammenhögsbadet och Korsavadbadet samt en samlad vy med diagram.

## Underlag: utfall och budget (2026-03-30)

För **badens utfall och budget per 2026-03-30** (datum i filnamnet: 26-03-30) finns ett separat dokument i repot:

**[`260330_badanalys.pdf`](./260330_badanalys.pdf)**

Använd PDF:en som referens för siffror och analys; kalkylappen är det interaktiva verktyget som kan komplettera det underlaget.
